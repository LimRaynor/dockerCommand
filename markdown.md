# 마크다운 사용법



## 기본
>
- 불릿리스트 -> 내용 맨앞에 동그란 말표시 남기는거
  -> **" + ",  " - ",  " * "** 이세가지를 "" **쌍따옴표 떼고  맨앞 내용**에 사용하면됨

>
- **제목**과 **내용**이 있다면 벨로그는 항상 **자동저장**되니 걱정 ㄴㄴ
  자동저장된 글은 우측상단 프로필 눌르면 **임시글**에 있음

>
- **태그**를 입력하여 나만의 모음집을 만들거나 나중에 참고하기 쉽게
  하나의 태그로 관리하면 편함!!

>
- 파워링크 만드는 것은 내가 사용할 링크 제목등을 임의로 만든후 **제목에 "[ ]"** 를 씌워주고, 그**뒤에 "( )" 를달고** 괄호 안에 내가 넣고싶은 **링크 주소**를 넣으면된다
  ex) **예시 -> [ 예시 ] +( 링크 주소 )**

>
- 글 내용 작성시 #의 개수만큼 글 안에서 소제목 부여가능!
  ex) 첫줄에 **" # 입력후 스페이스바 "** 하면 소제목 탄생!
  ex2) **#** 의 개수만큼 소제목안의 부제목도 부여가능!

>
- 내용이 끝나거나 새로운 내용을적을때 " _ " 를 이용한 마침줄 긑기
  ex) **___ 와같이 내용 맨앞에 3개**를 연달아 입력하면 밑과 같은 마침줄이 생김
___

>
- ~~취소선 텍스트~~ <-- 처럼 단어 **양옆에 ~~ 을 추가**하면 중요한 내용을 강조 가능!
  ex) 내용 -> ~~ 이걸 양옆에 써주면 -> ~~내용~~

>
- *이탤릭체 텍스트*<-- 처럼 단어 **양옆에 * 을 1개씩 추가**하면 중요한 내용을 강조 가능!
  ex) 내용 ->  *이걸 양옆에 써주면 * <-- 요렇게 기울여져 보이게됨




>
- **볼드체** <-- 처럼 단어 **양옆에 * 을 2개씩 추가하**면 중요한 내용을 강조 가능!
  ex) 내용 -> * * 이걸 양옆에 써주면 -> **내용**

## 마크다운 사용법





> 마크다운 (Markdown) 사용법




위처럼 **" > " 첫줄**에 넣고 글을 작성하면 내가 넘기지 않을경우 마크다운으로 작성됨

### 마크다운 안의 마크다운 인용문

> 1단

>> 2단


>>> 3단


>>>> 4단


처럼 마크다운 " > " 을 입력후 추가로 " > " 필요한만큼 추가해주면 된다
ex) 1.> 2.>> 3.>>> 4.>>>> ...

## 입력 코드 방식
벨로그를 쓸때 내가쓰는 프로그램의 코드를쓰면 남들은
```python
def hello_velog():
    print("Welcome to Velog!")
```
이렇게 **코드스럽게** 써지는걸 봤을텐데


def hello_velog():
print("Welcome to Velog!")


내가 막상 벨로그에 코드를 쓰면 이런식으로 밋밋하게 써질거야
어떻게 **코드스럽게** 쓰냐면

``` 
ex) ``` [java] <-- 점옆에 [] 안에 내가쓰는 언어타입
public class helloWorld{
	public void main(String[] args) {
		System.out.println("Hello World");
	}
}```  <--- 이 점 세개를 코드 맨위와 맨아래에 적어주기!
```

**이런식으로 코드작성이 가능해진다
위코드를 밑에 써보게 되면**
``` java 
public class helloWorld{
	public void main(String[] args) {
		System.out.println("Hello World");
	}
}
```  
이렇게 내가알던 코드형식으로 작성가능!!

그러므로 내가사용하는 언어타입에 맞게 [ ] 안에 선언하면
**코드스럽게** 내용에 적을 수 있게 된다.

## 체크박스
- 주로 " _ " 기호를 " [ ] " 대괄호 안에 넣어 체크박스를 만들어 사용한다
- [ ]  <-- 완성된 박스




> **이해를돕기위해 1을 입력**
- [ 1] 체크되지 않은 박스
- [x1] 체크된 박스

> **1을 지운 결과물**
- [ ] 체크되지 않은 박스
- [x] 체크된 박스
